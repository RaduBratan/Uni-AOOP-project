## Details
This is a project I created for my Advanced Object-Oriented Programming class at UniBuc FMI. The theme of the project (which I chose myself) is "Game Reviews Management System".

## Requirements for part 1

#### Have at least 10 actions:
- create developer account
- create user account
- show developer account
- show user account
- add game to dev
- add review to game
- show game of dev
- show review of game
- remove offensive game of dev
- remove offensive review of game

#### Use at least 8 classes (I created 19):
01. GameData
02. GameDatabase
03. GameFactory
04. GameSingleton
05. ReviewData
06. ReviewDatabase
07. ReviewFactory
08. ReviewSingleton
09. UserData
10. UserDatabase
11. UserFactory
12. UserSingleton
13. DeveloperData
14. DeveloperDatabase
15. DeveloperFactory
16. DeveloperSingleton
17. CSVService
18. MainService
19. Main
